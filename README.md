GenPEAR
=======
Use simple package.ini file to generate hard PEAR package.xml!

Example spec file:

    [package]
    name = GenPEAR
    desc = package description
    homepage = http://your.web.com
    license = MIT

    [author]
    name = Yo-An Lin
    email = cornelius.howl@gmail.com

    [require]
    php = >=5.3
    symfony/process = >=1.1
